# Demo guide

1. Start the API and web app as described in the README; no API key is needed.
2. In **Agent Demo**, choose a synthetic example, optionally select a `SYN-*` profile, and run the workflow.
3. Inspect routing, timeline, citations, JSON, missing context, safety block, and human-review messages.
4. In **Evaluation**, run all built-in synthetic cases, review summary rates, filter categories or failures, and expand expected/actual JSON.
5. Use **Architecture** and **About** to explain boundaries.

Do not paste real names, contact details, records, policies, prompts, endpoints, or confidential information. Metrics are local deterministic test results, not production claims.

# Data privacy and intellectual property

This public repository accepts only fictional or synthetic examples. Acme Corporation is fictional, as are every named person, identifier, location, policy, benefit, and question included here.

Do not submit real employee or candidate data, confidential prompts, internal policies or workflows, private endpoints, screenshots, credentials, secrets, customer information, or code copied from an employer or client. Remove metadata and review fixtures before contribution. Use `.env` for local settings and never commit it.

The phase-1 mock provider sends no content to an external model. Future integrations must be opt-in, document data flows and retention, minimize transmitted data, and include human review. Report suspected exposure according to `SECURITY.md` rather than in a public issue.

# Safety boundaries

The default Mock Agent does not make employment decisions. It detects prompt-injection phrases before routing and does not follow instructions to override rules or reveal records.

Human review is required for termination, discipline, hiring ranking, promotion, compensation adjustment, medical judgment, discriminatory judgment, legal conclusions, and unauthorized access to another employee's private information. The Agent may summarize fictional public policy, state missing information, offer a neutral ticket summary, and refer the request to an authorized HR professional.

Rules are illustrative safeguards, not comprehensive security controls. This project is not a legal, HR, or employment-decision tool and must not process real personal data.

Evaluation uses only repository-owned synthetic fixtures. The run endpoint has no request model, deliberately preventing user-supplied employee records. Citation IDs must exist in the fictional source manifest, and missing policy knowledge produces an explicit no-invention response.
